# Amazon Q

Amazon Q is a generative AI–powered assistant designed specifically for work. It can be tailored to your business, helping employees get answers to questions, solve problems, generate content, and take actions using the data and expertise found in your company's systems.

## Key Capabilities

- **Amazon Q Business:** Empowers employees to get answers, summarize documents, and generate content based on enterprise data. It connects to 40+ popular enterprise data sources (e.g., S3, SharePoint, Salesforce).
- **Amazon Q Developer:** Helps developers and IT pros with coding, testing, upgrading applications, and troubleshooting AWS infrastructure.
- **Amazon Q in Connect:** Helps customer service agents provide better support by recommending responses and actions based on real-time conversations.
- **Amazon Q in QuickSight:** Enables business users to build dashboards and perform data analysis using natural language.

## Enterprise Features

- **Security & Privacy:** Amazon Q respects existing permissions and access controls. It does not use your data to train the underlying models.
- **Admin Controls:** Admins can enable/disable specific capabilities and manage data source connections.
- **Content Filtering:** Built-in safeguards to ensure professional and relevant responses.

## Use Cases

- **Internal Knowledge Search:** Employees asking "What is our company policy on X?" and getting an answer based on internal HR documents.
- **Code Transformation:** Upgrading Java applications from older versions to newer ones using Amazon Q Code Transformation.
- **AWS Troubleshooting:** Asking "Why is my Lambda function failing?" and getting architectural advice.

## Exam Tips (SAP-C02)

- **Integration:** Amazon Q Business is the go-to for "enterprise search" or "AI assistant for employees" scenarios.
- **Permissions:** It honors the permissions defined in the connected data sources (e.g., if a user doesn't have access to a folder in S3, Amazon Q won't show them answers from that folder).
- **Comparison:** Unlike Bedrock (which is a platform to *build* AI apps), Amazon Q is a *ready-to-use* AI assistant.
