# AWS Machine Learning Services (SAA-C03 Focus)

### Amazon Comprehend
- Fully managed NLP service for text analysis (sentiment, entities, language, key phrases).
- Supports custom entity recognition and classification.
- Integrates with S3, Lambda, and other AWS services.
- Use cases: customer feedback, document analysis, content moderation.

**Sample Questions:**
1. Which AWS service is best for extracting sentiment and entities from customer reviews? (A: Amazon Comprehend)
2. How can you automate content moderation for large volumes of text in S3? (A: Use Amazon Comprehend with Lambda triggers)

### Amazon Forecast
- Managed service for time-series forecasting (sales, inventory, demand).
- Uses machine learning to generate accurate forecasts.
- Integrates with S3 for data input/output.
- No ML expertise required; automates model selection and tuning.

**Sample Questions:**
1. Which AWS service should you use to predict future sales based on historical data? (A: Amazon Forecast)
2. What is a key benefit of Amazon Forecast for business users? (A: No ML expertise required)

### Amazon Fraud Detector
- Detects online fraud using ML models (e.g., new account fraud, transaction fraud).
- Integrates with S3, Lambda, and API Gateway.
- Customizable rules and models for specific business needs.
- Use cases: e-commerce, financial transactions, account creation.

**Sample Questions:**
1. Which AWS service helps detect fraudulent online transactions? (A: Amazon Fraud Detector)
2. How can you customize fraud detection for your business? (A: Create custom rules/models in Amazon Fraud Detector)

### Amazon Kendra
- Intelligent search service powered by ML for enterprise data.
- Supports natural language queries and relevance ranking.
- Integrates with S3, SharePoint, databases, and more.
- Use cases: document search, knowledge management, intranet search.

**Sample Questions:**
1. Which AWS service enables natural language search across enterprise documents? (A: Amazon Kendra)
2. How does Amazon Kendra improve search accuracy? (A: Uses ML for relevance ranking)

### Amazon Lex
- Conversational AI for chatbots and voice assistants.
- Supports speech recognition and natural language understanding.
- Integrates with Lambda, Connect, and other AWS services.
- Use cases: customer support bots, IVR systems, virtual assistants.

**Sample Questions:**
1. Which AWS service is used to build chatbots with speech and text input? (A: Amazon Lex)
2. How can you automate customer support using AWS? (A: Deploy a chatbot with Amazon Lex)

### Amazon Polly
- Text-to-speech service with lifelike voices.
- Supports multiple languages and custom lexicons.
- Integrates with S3, Lambda, and Connect.
- Use cases: voice-enabled apps, accessibility, IVR systems.

**Sample Questions:**
1. Which AWS service converts text to natural-sounding speech? (A: Amazon Polly)
2. How can you add voice output to a web application? (A: Use Amazon Polly API)

### Amazon Rekognition
- Image and video analysis (label detection, face recognition, moderation).
- Supports custom labels and celebrity recognition.
- Integrates with S3, Lambda, and other AWS services.
- Use cases: security, media analysis, content moderation.

**Sample Questions:**
1. Which AWS service is best for detecting objects and faces in images? (A: Amazon Rekognition)
2. How can you automate video content moderation? (A: Use Amazon Rekognition with Lambda)

### Amazon SageMaker
- End-to-end ML platform: build, train, deploy models at scale.
- Supports Jupyter notebooks, built-in algorithms, and managed infrastructure.
- Integrates with S3, Lambda, and other AWS services.
- Use cases: custom ML solutions, data science workflows, production deployment.

**Sample Questions:**
1. Which AWS service provides a managed platform for building and deploying ML models? (A: Amazon SageMaker)
2. How can you automate ML model training and deployment? (A: Use Amazon SageMaker pipelines)

### Amazon Textract
- Extracts text, tables, and forms from scanned documents.
- Supports OCR and structured data extraction.
- Integrates with S3, Lambda, and other AWS services.
- Use cases: document automation, compliance, data entry.

**Sample Questions:**
1. Which AWS service extracts structured data from scanned PDFs? (A: Amazon Textract)
2. How can you automate form data extraction in AWS? (A: Use Amazon Textract with Lambda)

### Amazon Transcribe
- Automatic speech-to-text transcription service.
- Supports multiple languages and custom vocabulary.
- Integrates with S3, Lambda, and other AWS services.
- Use cases: call center analytics, video captioning, meeting notes.

**Sample Questions:**
1. Which AWS service converts audio files to text? (A: Amazon Transcribe)
2. How can you generate captions for recorded webinars? (A: Use Amazon Transcribe)

### Amazon Translate
- Neural machine translation for real-time and batch translation.
- Supports dozens of languages.
- Integrates with S3, Lambda, and other AWS services.
- Use cases: website localization, document translation, multilingual apps.

**Sample Questions:**
1. Which AWS service provides real-time translation for web applications? (A: Amazon Translate)
2. How can you automate document translation in AWS? (A: Use Amazon Translate with S3 and Lambda)
