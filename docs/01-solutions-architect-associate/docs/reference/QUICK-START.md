## 💡 Study Tips

### ✅ Do This
- Start with Module 01 (builds foundation)
- Read each section title first (overview)
- Focus on tables and decision trees
- Memorize mnemonics (they work!)
- Complete practice questions immediately
- Review "Common Mistakes" sections
- Take short breaks (5 min every hour)

### ❌ Don't Do This
- Skip modules (they build on each other)
- Just read passively (engage with material)
- Ignore practice questions
- Cram everything in one sitting
- Skip the "5-Minute Master Review"
- Forget to review weak areas

---

## 🎯 Daily Study Routine

### Recommended Flow (2-3 hour session)
```
1. Review previous day (10 min)
   └─ Quick scan of yesterday's material

2. New module (60-90 min)
   └─ Read FAST-LEARN.md file
   └─ Take notes on tables/mnemonics
   
3. Practice (20-30 min)
   └─ Answer practice speedrun questions
   └─ Review wrong answers

4. Break (10 min)
   └─ Walk, stretch, hydrate

5. Review (15-20 min)
   └─ Re-read "5-Minute Master Review"
   └─ Create flashcards for weak areas

6. Plan next session (5 min)
   └─ Note what to study tomorrow
```
