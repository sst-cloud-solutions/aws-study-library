## 🎨 Folder Categories Explained

### 📁 navigation/ - Your Starting Point
**Purpose:** Help you navigate and understand all study materials  
**When to use:** First visit, finding documents, making decisions  
**Total files:** 10  
**Time investment:** 5 min to 30 min depending on document

**Key files:**
- **00-CATEGORIZATION-INDEX.md** - Most comprehensive guide
- **CATEGORY-QUICK-REFERENCE.md** - Fastest lookup
- **DOCUMENT-SELECTION-MATRIX.md** - Help choosing what to read
- **POCKET-GUIDE.md** - Save to phone for mobile access

---

### 📁 quick-reference/ - Exam Day Essentials
**Purpose:** Last-minute review and memorization  
**When to use:** Night before exam, morning of exam, quick refreshers  
**Total files:** 2  
**Time investment:** 15-20 min each

**Contains:**
- **ULTRA-QUICK-REFERENCE-CARD.md** - Top 10 must-memorize rules
- **MEMORY-CARDS.md** - Visual mnemonics and memory aids

---

### 📁 master-guides/ - Big Picture Overview
**Purpose:** Understand overall progress and learn key concepts  
**When to use:** Beginning study, understanding status, learning patterns  
**Total files:** 2  
**Time investment:** 45-90 min each

**Contains:**
- **COMPLETE-EXAM-REVIEW.md** - Performance analysis, study plan
- **ALL-TESTS-MASTER-QUICK-REFERENCE.md** - Top 135+ concepts from all tests

---

### 📁 complete-reviews/ - Deep Learning
**Purpose:** Detailed analysis of each practice test  
**When to use:** First review after taking a test, deep learning sessions  
**Total files:** 5 (one per test)  
**Time investment:** 2-4 hrs each (14 hrs total)

**Contains:**
- Full test reviews with detailed explanations
- Question analysis (without actual questions)
- Domain performance breakdowns
- Learning notes and diagrams

---

### 📁 condensed-reviews/ - Fast Review
**Purpose:** Quick refresher of key learnings  
**When to use:** Second review, refreshers, time-constrained study  
**Total files:** 5 (one per test)  
**Time investment:** 20-40 min each (2.3 hrs total)

**Contains:**
- Essential information only
- Key concepts without verbose explanations
- Quick reference for mistakes
- Domain scores and priorities

## 📋 File Organization Benefits

### ✅ Advantages of Folder Structure:

1. **Clearer Organization**
   - Files grouped by purpose
   - Easy to find what you need
   - Reduced visual clutter

2. **Better Navigation**
   - Logical folder names
   - Clear category separation
   - Intuitive browsing

3. **Efficient Study Planning**
   - Focus on one folder at a time
   - Progress tracking per category
   - Clear study paths

4. **Scalability**
   - Easy to add new test reviews
   - Simple to add new categories
   - Maintainable structure

5. **Mental Model**
   - Categories match study phases
   - Folder names indicate content
   - Natural progression path

## 💡 Folder Navigation Tips

### Tip 1: Bookmark These Locations
- `exam-reviews/` - Root folder
- `exam-reviews/navigation/` - Always start here
- `exam-reviews/quick-reference/` - Exam day essentials

### Tip 2: Progressive Learning Path
1. Start with `navigation/` to understand structure
2. Move to `master-guides/` for overview
3. Use `complete-reviews/` for deep learning
4. Use `condensed-reviews/` for reinforcement
5. Use `quick-reference/` for final prep

### Tip 3: Folder = Study Phase
- **navigation/** = Orientation phase
- **master-guides/** = Understanding phase
- **complete-reviews/** = Learning phase
- **condensed-reviews/** = Reinforcement phase
- **quick-reference/** = Mastery phase

### Tip 4: Time-Based Selection
- \< 30 min? → `quick-reference/`
- 30-90 min? → `master-guides/` or `condensed-reviews/`
- 2+ hrs? → `complete-reviews/`
- Unsure? → `navigation/`
