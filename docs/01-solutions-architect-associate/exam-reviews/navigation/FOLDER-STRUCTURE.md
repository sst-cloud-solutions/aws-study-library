## 🎯 File Naming Convention

### Pattern Analysis

```
Navigation Files:
- ALL-CAPS-WITH-DASHES.md
- 00-PREFIX for main index
- Descriptive names (INDEX, REFERENCE, NAVIGATION)

Test Reviews:
- Practice-Test-[Number]-Review[-Condensed].md
- Numbers: 1-5
- Optional: -Condensed suffix for shorter versions

Special Files:
- ULTRA-QUICK-REFERENCE-CARD.md (pre-exam)
- MEMORY-CARDS.md (visual learning)
- ALL-TESTS-MASTER-QUICK-REFERENCE.md (comprehensive)
```

## 🎓 Pro Tips for Navigating

### 1. **Bookmark These 3 Files:**
   - 00-CATEGORIZATION-INDEX.md (comprehensive guide)
   - CATEGORY-QUICK-REFERENCE.md (quick lookup)
   - ULTRA-QUICK-REFERENCE-CARD.md (exam day)

### 2. **File Naming Logic:**
   - `00-` prefix = Start here
   - `ALL-TESTS` = Combines all 5 tests
   - `COMPLETE` = Overall summary
   - `ULTRA-QUICK` = Shortest, most critical
   - `-Condensed` suffix = Shorter version

### 3. **When Files Link Together:**
   - Navigation files → Study materials
   - Master guides → Individual test reviews
   - Complete reviews → Condensed versions
   - All point back to 00-CATEGORIZATION-INDEX.md

### 4. **Color Coding (if viewing with compatible markdown viewer):**
   - 🚀 = Use before exam
   - 📚 = Comprehensive learning
   - 📝 = Detailed reviews
   - ⚡ = Fast versions
   - 🧭 = Navigation helpers
